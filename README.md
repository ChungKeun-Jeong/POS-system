## POS-system
스프링 프레임워크 프로젝트 - POS 시스템 
<br><br><br>

## 기능

- 회원가입
- 로그인
- 제품 판매
- 제품 입고
- 재품 정보 및 재고 확인
- 통계
<br><br><br>

<br>
